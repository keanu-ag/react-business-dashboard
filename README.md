# React Business Insights Dashboard

## Description
A frontend business dashboard built with React to visualize key metrics and reports, inspired by real corporate applications.

## Features
- KPI cards
- Interactive charts
- Data tables with filters
- Mock API integration
- Context API for state management

## Tech Stack
- React (JavaScript)
- Context API
- Vite
- Chart.js
- CSS

## How to Run
npm install
npm run dev
